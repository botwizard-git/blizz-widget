Blizz Widget - Remote URL Example

Welcome to the Remote Example

This page demonstrates loading the EnterpriseBotBlizz widget from a remote CDN URL.

Widget loaded from: https://blizz.botwizard.ch/blizz.js

Integration Code

To embed this widget on any website, add the following code:

<!-- Widget Container -->
<div id="enterprisebot-blizz-parent"></div>

<!-- Load Widget Script -->
<script src="https://blizz.botwizard.ch/blizz.js"></script>
