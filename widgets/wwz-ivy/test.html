WWZ Ivy Chatbot - Test Page

Willkommen auf der Testseite

Diese Seite demonstriert das WWZ Ivy Chatbot-Widget. Klicken Sie auf die blaue Roboter-Blase in der unteren rechten Ecke, um den Chatbot zu öffnen.

Funktionen

  • Begrüssungsbildschirm mit Nutzungsbedingungen
  • Chat-Interface mit Nachrichtenblasen
  • Kopieren und Text-zu-Sprache für Bot-Nachrichten
  • Spracheingabe (Spracherkennung)
  • Vorschlags-Buttons
  • Feedback-Bewertung (1-5)
  • Chat-Transkript herunterladen
  • Vollbildmodus
  • HTML-Rendering für Bot-Antworten

API-Steuerung

Sie können das Widget programmatisch über die window.WWZIvy.Main API steuern:

Verfügbare Methoden:

  • WWZIvy.Main.expand() - Widget öffnen
  • WWZIvy.Main.collapse() - Widget schliessen
  • WWZIvy.Main.startNewSession() - Neuen Chat starten
  • WWZIvy.Main.sendMessage(text) - Nachricht senden
  • WWZIvy.Main.getVersionInfo() - Versionsinfo abrufen
  • WWZIvy.Main.getState() - Aktuellen Status abrufen

Einbindungsanleitung

Um diesen Chatbot auf Ihrer Website einzubinden, fügen Sie folgenden Code hinzu:

<div id="wwz-ivy-parent"></div>

<script src="wwz-ivy.js"></script>

Das Widget wird automatisch initialisiert und erscheint als schwebende Blase in der unteren rechten Ecke.
